## 下载和安装

### Windows 系统

#### 系统要求
- Windows 10 或 Windows 11（64位）
- 至少 200 MB 可用磁盘空间
- 稳定的网络连接

#### 下载步骤

1. 访问官网：[https://dj.siumingho.dpdns.org](https://dj.siumingho.dpdns.org)
2. 点击顶部导航栏的 **"下载客户端"** 按钮
3. 在下载页面选择 **Windows** 选项卡
4. 点击 **"开始下载"** 按钮
5. 等待下载完成（文件大小约 50 MB）

#### 安装步骤

1. 找到下载的 `LogVPN_Setup.exe` 文件
2. 双击运行安装程序
3. 如果出现 Windows 安全提示，点击 **"更多信息"** → **"仍要运行"**
4. 按照安装向导提示完成安装：
   - 阅读并同意许可协议
   - 选择安装位置（默认：`C:\Program Files\LogVPN`）
   - 选择是否创建桌面快捷方式
   - 点击 **"安装"** 开始安装
5. 安装完成后，勾选 **"启动 LogVPN"** 并点击 **"完成"**

## 首次使用

### 1. 登录账号

首次启动 LogVPN 时，会自动打开登录页面：

1. 点击 **"使用账号登录"** 按钮
2. 浏览器会自动打开 LogVPN 官网登录页面
3. 输入您的邮箱和密码登录
4. 登录成功后，浏览器会自动跳转并关闭
5. LogVPN 客户端会自动完成登录

> **提示**：如果您还没有账号，请先在官网注册：[https://dj.siumingho.dpdns.org/register](https://dj.siumingho.dpdns.org/register)

### 2. 选择节点

登录成功后，LogVPN 会自动获取可用节点列表：

1. 在主界面的节点列表中查看所有可用节点
2. 节点按地区分类（如：香港、美国、日本、新加坡等）
3. 每个节点显示以下信息：
   - 节点名称
   - 地理位置
   - 延迟（ping 值）
   - 负载状态
4. 点击任意节点即可选中

### 3. 一键连接

选择节点后，点击主界面的 **"连接"** 按钮：

1. 客户端会自动配置代理设置
2. 连接成功后，按钮变为 **"断开"**
3. 主界面显示实时流量统计：
   - 上传速度
   - 下载速度
   - 已用流量
   - 连接时长

## 功能介绍

### 1. 自动登录

- LogVPN 使用 OAuth 2.0 认证，无需手动输入账号密码
- 登录状态会自动保存，下次启动无需重新登录
- 支持多设备登录（需要在官网开通多设备权限）

### 2. 节点管理

#### 节点列表
- 自动从服务器获取最新节点列表
- 节点信息实时更新
- 支持按地区、延迟、负载排序

#### 延迟测试
- 点击节点旁边的 **"测速"** 按钮
- 自动测试节点延迟（ping 值）
- 延迟越低，连接速度越快

#### 智能推荐
- LogVPN 会根据您的网络状况自动推荐最佳节点
- 推荐节点标记为 **"推荐"** 标签
- 优先选择推荐节点以获得最佳体验

### 3. 流量统计

#### 实时流量
- 主界面显示当前上传/下载速度
- 以图表形式展示流量变化趋势
- 支持切换显示单位（KB/s、MB/s）

#### 流量统计
- 查看今日已用流量
- 查看本月已用流量
- 查看剩余流量配额
- 流量数据自动同步到服务器

### 4. 设备指纹防共享

- LogVPN 使用设备指纹技术识别设备
- 一个账号同时只能在一台设备登录
- 新设备登录时，旧设备会自动断开连接
- 防止账号共享，保护账号安全

> **注意**：如果您需要更换设备，请先在旧设备上退出登录，然后在新设备上登录。

### 5. 自动更新

- LogVPN 启动时会自动检查更新
- 如果有新版本，会弹出更新提示
- 点击 **"立即更新"** 自动下载并安装
- 更新完成后自动重启客户端

### 6. 系统代理

- LogVPN 支持全局代理和 PAC 模式
- **全局代理**：所有网络流量通过 VPN
- **PAC 模式**：根据规则自动选择是否使用 VPN
- 在设置中切换代理模式

## 常见问题

### Q1: 无法连接到节点

**可能原因**：
- 网络连接不稳定
- 节点负载过高
- 防火墙阻止连接

**解决方案**：
1. 检查您的网络连接是否正常
2. 尝试更换其他节点
3. 检查防火墙设置，允许 LogVPN 通过
4. 重启 LogVPN 客户端

### Q2: 连接后无法访问网站

**可能原因**：
- 代理设置未生效
- DNS 解析问题
- 节点故障

**解决方案**：
1. 检查系统代理设置是否正确
2. 尝试刷新 DNS 缓存：
   ```
   Windows: ipconfig /flushdns
   ```
3. 更换其他节点重新连接
4. 重启浏览器

### Q3: 流量统计不准确

**可能原因**：
- 网络延迟导致数据同步延迟
- 客户端缓存未刷新

**解决方案**：
1. 等待几分钟让数据自动同步
2. 点击 **"刷新"** 按钮手动同步
3. 重启客户端

### Q4: 提示"设备验证失败"

**可能原因**：
- 您的账号已在其他设备登录
- 设备指纹发生变化（如重装系统）

**解决方案**：
1. 在其他设备上退出登录
2. 联系客服解绑设备
3. 等待 10 分钟后重新登录

### Q5: 下载速度很慢

**可能原因**：
- 节点负载过高
- 您的网络带宽限制
- 连接的节点距离较远

**解决方案**：
1. 更换负载较低的节点
2. 选择距离您较近的节点（如同一国家或地区）
3. 升级您的网络套餐
4. 联系客服报告问题

### Q6: 无法自动更新

**可能原因**：
- 网络连接问题
- 更新服务器维护
- 权限不足

**解决方案**：
1. 检查网络连接是否正常
2. 稍后再试
3. 以管理员身份运行 LogVPN
4. 手动下载最新版本安装
